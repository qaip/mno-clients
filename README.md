# MNO Clients

## Description
A Kivy app for mobile network operator to view and manage their clients.

## Features
- Adding a new client
- Searching for clients
- Deleting clients

## Development

Install the dependencies:
```bash
poetry install
```

Run the database:
```bash
docker compose up
```

Run the app:
```bash
poetry run app
```
